Simple by design

Four steps. Zero effort.

KaggleBoost handles the entire pipeline automatically. You just watch your profile grow.

🔍
01

Scrape

Pulls live, openly licensed data from 5 high-demand sources: World Bank, NASA, Yahoo Finance, Our World in Data, and India Open Data.

🧹
02

Clean

Auto-fixes types, removes duplicates, detects PII, strips nulls, and formats column names. Every dataset arrives clean and ready to use.

Package

Generates a professional README, summary statistics, column descriptions, and a valid dataset-metadata.json — the way top datasets look.

04

Upload

Publishes directly to your Kaggle profile via the Kaggle API. Creates new datasets or pushes fresh versions. Runs every 3 days on autopilot.

FAQ

Common questions


Is this against Kaggle's terms of service? +
No. KaggleBoost uploads datasets you own using the official Kaggle API — exactly the same as uploading manually. All scraped data is openly licensed (CC-BY, Public Domain, ODbL). You become the legitimate publisher.
How long does it take to see results? +
Most users see their first datasets live within 5 minutes of setup. Profile views typically increase within the first week as datasets get indexed by Kaggle's search engine.
Do I need to pay for hosting? +
No. KaggleBoost runs entirely on GitHub Actions, which gives you 2,000 free minutes per month — more than enough. Zero server costs.
Will my Kaggle API key stay private? +
Yes. Your API key is stored as a GitHub Secret — encrypted at rest, never exposed in logs or code. KaggleBoost never sees or stores your credentials.
